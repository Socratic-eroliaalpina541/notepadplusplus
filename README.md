# Notepad++

**Notepad++** is a free source code editor and Notepad replacement that supports several programming languages and offers a wide range of features for developers and power users.

## Key Features
- Syntax highlighting for many programming languages
- Auto-completion and code folding
- Multi-tab editing and split screen
- Plugin support for extended functionality
- Search and replace with regular expressions
- Lightweight and fast performance

## Getting Started
1. **Download** the latest version using the button above.
2. **Extract** the archive.
3. **Run** `notepad++.exe`.
4. Open your files or start coding.

**Note:** This version may include additional plugins or custom settings.
